# Zomato Data Analysis Project
## Project Overview:
This project analyzes Zomato restaurant data to uncover patterns in customer behavior, ratings, and ordering preferences. The goal is to understand restaurant popularity, customer spending, and order trends to provide actionable insights for restaurants and Zomato.

## Key Insights:
1. Restaurant Type Preference:
Majority of customers order from dining restaurants.
  
2. Votes by Restaurant Type:
- Dining restaurants received the highest number of votes from customers.

3. Ratings Distribution:
- Most restaurants have ratings between 3.5 to 4.

4. Average Spending by Couples:
- Couples generally prefer restaurants with an approximate cost of â‚¹300 for two.

5. Online vs Offline Ratings:
- Online orders tend to receive higher ratings compared to offline orders.

6. Order Mode by Restaurant Type:
- Dining restaurants mainly receive offline orders, while cafes primarily receive online orders.
- This insight can help Zomato optimize offers and promotions.

## Visualizations:
- Countplots: Show the distribution of restaurant types and spending.
- Line plots: Total votes by restaurant type.
- Histograms: Ratings distribution.
- Boxplots: Ratings by online/offline order mode.
- Heatmaps: Order mode distribution across restaurant types.

## Requirements:
- Python 3.x
- Libraries:
  - pandas
  - numpy
  - matplotlib
  - seaborn

Make sure Zomato data.csv is in the working directory.
